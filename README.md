<div id="usage_ar"></div>

### 🚀 طريقة الاستخدام

1. **التشغيل**: قم بفتح ملف `UsernameGeneratorAR_EN.exe`.
2. **إعداد الطول**: حدد طول اسم المستخدم المطلوب (من 4 إلى 30 حرف).
3. **تخصيص المكونات**: اختر تضمين أحرف كبيرة، صغيرة، أو أرقام.
4. **اختيار الرموز**: حدد الرموز المسموحة (`_` أو `.`) ومكان ظهورها من القائمة المنسدلة.
5. **التوليد والنسخ**: اضغط "توليد اسم مستخدم" ثم اضغط على أيقونة اللوحة 📋 لنسخ الاسم.
6. **تغيير اللغة**: اضغط على زر اللغة في الأعلى للتحويل فوراً بين العربية والإنجليزية.

## English

<div dir="ltr">

### Overview

**Username Generator** is a sophisticated desktop application designed to create unique, catchy, and professional usernames for social media, gaming, or professional accounts. It features advanced placement logic for symbols and numbers to ensure high-quality results.

---

<div id="features_en"></div>

### ✨ Features

| Feature | Description |
|--------|-------------|
| 👤 **Unique Generation** | Creates non-repetitive and attractive usernames |
| 📍 **Smart Placement** | Control symbol positions (Start, End, Middle, or Between letters) |
| 🧠 **Logic Mode** | "Words then Numbers" option for natural-looking usernames |
| 🛡️ **Built-in Protection** | Advanced anti-debug and anti-VM layers for security |
| 📋 **Quick Copy** | One-click copy with a visual toast notification |
| 🌐 **Bilingual UI** | Seamless toggle between Arabic and English |
| 🎨 **Modern Design** | Eye-friendly dark theme using professional Cairo fonts |

<div id="usage_en"></div>

### 🚀 How to Use

1. **Launch**: Open `UsernameGeneratorAR_EN.exe`.
2. **Set Length**: Choose username length (4 to 30 characters).
3. **Customize**: Toggle Uppercase, Lowercase, or Numbers.
4. **Symbol Placement**: Select allowed symbols (`_` or `.`) and their position from the dropdown.
5. **Generate & Copy**: Click "Generate Username" and then the clipboard icon 📋 to copy.
6. **Toggle Language**: Use the language button at the top to switch UI language instantly.
